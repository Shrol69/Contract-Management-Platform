# Contract Management Platform

A modern **Contract Management Web Application** built using **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, and **Zustand** for state management.

Live Demo:  
https://contractflow-chi.vercel.app/

This project demonstrates real-world frontend architecture, dynamic routing, state management, and PDF generation in a production-ready setup.

---

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Zustand – State Management
- Lucide React – Icons
- jsPDF – PDF generation
- dom-to-image-more – DOM to image rendering

---

## Features

- Create and manage contracts  
- Blueprint (template) based contract creation  
- Dynamic contract routing using App Router  
- Contract lifecycle tracking:  
  Created → Approved → Sent → Signed → Locked  
- Real-time status updates  
- Contract preview before finalization  
- Download contract as PDF  
- Clean, responsive UI with smooth animations  
- Client-side state management  
- Deployed on Vercel  

---


## Installation & Local Setup

Follow these steps to run the project locally:

```bash
git clone https://github.com/your-username/Contract-Management-Platform.git
cd contract-app
npm install
npm run dev
Then open your browser and visit:

http://localhost:3000
```
